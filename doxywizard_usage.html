Doxywizard用法

Doxywizard是一个用于配置和运行doxygen的GUI前端。

注意，可以用要使用的配置文件作为参数启动doxywizard。

启动doxywizard时，它将显示主窗口(实际的外观取决于使用的操作系统)。

主窗口

窗口显示了配置和运行doxygen的步骤。第一步是选择一种配置doxygen的方法。

向导（wizard）
单击此按钮可快速配置最重要的设置，其余选项保留默认值。
专家（expert）
单击此按钮可访问所有配置选项。
加载（load）
单击此按钮从磁盘加载现有配置文件。

注意，您可以分步骤点击这几个按钮，例如，首先使用向导配置doxygen，然后通过专家微调设置。

配置好doxygen之后，需要将配置作为文件保存到磁盘。这第二步允许doxygen使用配置，并且有一个额外的优点，即可以重用配置，以便在以后的某个时间点以相同的设置运行doxygen。

由于一些配置选项可能使用相对路径，下一步是选择一个目录，从其中运行doxygen。这通常是源码树的根，大多数时候已经被正确地填充了。

保存配置文件并设置工作目录之后，就可以根据选择的设置运行doxygen了。按下“开始”按钮。一旦doxygen运行，你可以通过再次点击相同的按钮来取消它。doxygen生成的输出被捕获并显示在日志窗口中。doxygen完成后，可以将日志保存为文本文件。

向导对话框

如果您在步骤1中选择向导按钮，那么将出现一个带有许多选项卡的对话框。

向导对话框:项目设置

project选项卡中的字段说明了一切。一旦doxygen完成，目标目录就是查找结果的地方。Doxygen将把每种输出格式放在单独的子目录中。

向导对话框:操作模式

mode选项卡允许您选择doxygen查看源代码的方式。默认情况下，只查找已记录的内容。

您还可以选择doxygen如何显示结果。后者不会影响doxygen解析源代码的方式。

向导对话框:要产生的输出

您可以选择一个或多个doxygen应该产生的输出格式。对于HTML和 还有其他选项。

向导对话框:要生成的图

Doxygen可以生成许多图表。使用diagram选项卡，您可以选择要生成哪些图表。对于大多数图表来说， GraphViz 包的点工具是必须的(如果您使用MacOSX的二进制包，那么这个工具已经包括在内了)。

Expert 对话框

Expert对话框有许多选项卡字段，每个选项卡对应配置文件中的每个部分。每个选项卡字段包含许多行，每个行对应该部分中的每个配置选项。

输入组件的类型取决于配置选项的类型。

  • 对于每个布尔选项(配置文件中具有YES或NO的选项)，都有一个复选框。
  • 对于采用固定值集之一的项(如OUTPUT_LANGUAGE)，将使用组合框。
  • 对于从范围中获取整数值的项，使用自旋框。
  • 对于自由形式的字符串类型选项，有一个单行编辑字段
  • 对于接受字符串列表的选项，可以使用单行编辑字段，使用' + '按钮将该字符串添加到列表中，使用' - '按钮将所选字符串从列表中删除。还有一个' * '按钮，当按下时，它会用编辑字段中输入的字符串替换列表中选择的项。
  • 对于文件和文件夹条目，有特殊的按钮来启动文件选择对话框。
Expert对话框中的一些选项

点击对话框右下角的“帮助”按钮，然后点击项目，获取关于选项含义的附加信息。将出现带有附加信息的工具提示。

菜单选项

GUI前端有一个菜单，其中有几个有用的项

File menu
Open...（打开…）
这与主窗口中的“Load”按钮相同，允许从磁盘打开配置文件。
Save as..（另存为）
这与主窗口中的“Save”按钮相同，可用于将当前配置设置保存到磁盘。
Recent configurations（最近配置）
允许快速加载最近保存的配置。
Set as default...（设置为默认……）
将当前配置设置存储为下次启动GUI时使用的默认设置。您将被要求确认操作。
Reset...（重置）
将工厂默认值还原为要使用的默认设置。您将被要求确认操作。
